## 🚀 USAGE EXAMPLES

### Enable Distributed Mode
```
1. In Streamlit app: Check "🔄 Enable Distributed Multi-Agent Mode"
2. Select agents: Backend ✓, Frontend ✓, Test ✓
3. Optionally: Check "🤖 Enable Autonomous Self-Improvement"
4. Enter task: "Build a complete user authentication system"
5. Click: "🚀 Generate Draft"
```

Result: 
- Backend agent generates API and database code
- Frontend agent generates Streamlit UI
- Test agent generates comprehensive test suite
- All 3 agents execute in parallel (~8-10 seconds)
- Results automatically merged with conflict resolution
- Code validated and reviewed
- Optional improvement loop applied
- User approves before final commit

### Programmatic Usage
```python
orchestrator = TaskOrchestrator()
agent_registry = {
    "backend": BackendAgent(orchestrator).generate_code,
    "frontend": FrontendAgent(orchestrator).generate_code,
    "test": TestAgent(orchestrator).generate_code,
}
result = orchestrator.execute_distributed(agent_registry)
print(result["code"])  # Merged code from all agents
```

## 🔍 SYSTEM ARCHITECTURE

### Layered Architecture
```
┌─────────────────────────────────────┐
│    Streamlit UI (app.py)            │  ← User Interaction
└──────────────────┬──────────────────┘
│
┌──────────────────┴──────────────────┐
│   TaskOrchestrator (core/)          │  ← Workflow Coordination
└──────────┬───────────┬──────┬───────┘
│                      │          │
▼                      ▼          ▼
┌────────────┐   ┌────────────┐  ┌────────────┐
│ Planner    │   │ Programmer │  │ Reviewer   │
│ Agent      │   │ Agent      │  │ Agent      │
└────────────┘   └────────────┘  └────────────┘

┌────────────┬────────────┬────────────┬──────────────┐
│  Memory    │ Validator  │ GitHub     │ Configuration
│  Module    │ Module     │ Tool       │
└────────────┴────────────┴────────────┴──────────────┘
```

### Data Flow
```
User Task → Planner → Plan
         ↓
      Programmer → Code
         ↓
      Validator → Validation Result
         ↓
      Reviewer → Review Feedback
         ↓
      [Display in UI]
         ↓
   [User Approval]
         ↓
      GitHub Tool → Committed to Branch
```
